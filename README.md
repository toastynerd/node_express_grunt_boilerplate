Toasty Boilerplate
===========================
A Toasty fresh boiler plate designed for use with a client
side javascript framework. It contains:
  * Grunt/a Gruntfile
  * Express with watch
  * Sass
  * Mocha/Chai for testing
  * Super Agent for testing
  * Browserify
  * Bower
  * JSHint, to keeo me honest
